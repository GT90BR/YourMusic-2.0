YourMusic

YourMusic é um projeto desenvolvido durante a Imersão Dev com o objetivo de apresentar múltiplos gêneros musicais e alguns artistas importantes dentro de cada um deles. A proposta principal é criar uma experiência interativa, intuitiva e visualmente agradável para que o usuário explore novos estilos e descubra músicas e artistas de forma simples e organizada.

    Objetivo do Projeto

O objetivo do YourMusic é entregar uma plataforma que permita ao usuário navegar por diversos gêneros musicais, conhecer artistas relacionados e acessar links externos para ouvir mais sobre cada um deles. Tudo foi pensado para proporcionar:

Uma experiência do usuário clara e fluida;

Navegação intuitiva entre os gêneros e artistas;

Design moderno e estilizado, respeitando o esqueleto original ensinado na Imersão Dev;

Um ambiente agradável para descoberta musical.

    Funcionalidades

Exibição de uma lista de gêneros musicais;

Cards individuais com nome, descrição, ano e resumo dos artistas;

Links diretos para plataformas de música (como Spotify);

Layout responsivo e estilização avançada para melhorar a experiência visual.

    Tecnologias Utilizadas

HTML5

CSS3 (com estilização aprimorada e design próprio)

JavaScript (base do projeto da Imersão Dev)

      Estrutura do Projeto:

O projeto segue a estrutura original fornecida pela Imersão Dev, mantendo:

A lógica base das buscas e listagens;

O esqueleto obrigatório das aulas;

A separação adequada entre HTML, CSS e JavaScript.

     E complementa com:

Estilizações personalizadas;

Melhorias na apresentação do conteúdo.

Como Executar:

Baixe ou clone este repositório;

Abra o arquivo index.html no seu navegador;

Explore os gêneros musicais e artistas!
